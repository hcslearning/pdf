= PDFs

== Agregar Outline

. Descargar metadatos
+
[source, bash]
----
pdftk libro.pdf dump_data output meta.txt
----
+
. Editar los metadatos, debajo de "NumberOfPages" usar: BookmarkBegin, BookmarkTitle, BookmarkLevel y BookmarkPageNumber para generar el Ã­ndice. Ejemplo:
+
[source]
----
NumberOfPages: 438
BookmarkBegin
BookmarkTitle: Chapter 1 - Introducing Message Queuing
BookmarkLevel: 1
BookmarkPageNumber: 19
BookmarkBegin
BookmarkTitle: Chapter 2 - System.Messaging
BookmarkLevel: 1
BookmarkPageNumber: 72
----
+
. Generar el PDF con outline:
+
[source, bash]
----
pdftk libro.pdf update_info meta.txt output libro_con_indice.pdf
----
